# 자바스크립트 유틸 CMS 대시보드 기획서

## 1. 목적
- 다양한 자바스크립트 유틸리티 기능(문자열 변환, 날짜 계산, 인코딩/디코딩 등)을 웹에서 쉽게 사용할 수 있도록 제공
- 관리자/사용자가 직관적으로 접근할 수 있는 대시보드 형태의 UI 제공

## 2. 주요 기능
- 유틸리티별 메뉴 제공(문자열, 숫자, 날짜, 인코딩 등)
- 각 유틸리티별 입력/출력 폼 및 결과 미리보기
- 최근 사용 내역, 즐겨찾기 기능
- 반응형 UI, 다크모드 지원

## 3. 타겟 사용자
- 개발자, 기획자, 일반 사용자 등 자바스크립트 유틸을 웹에서 빠르게 사용하고 싶은 모든 사용자

## 4. 메뉴 목록

### 현재 구현된 메뉴
1. **문자열**
   - 문자열 변환 (대/소문자, 공백/특수문자 제거, 자르기/합치기)
   - 문자열 저장 (로컬스토리지)
   - 숫자 변환 (진수 변환, 반올림/올림/내림)
   - 날짜/시간 (포맷 변환, 차이 계산, 타임스탬프)

2. **데이터/객체**
   - JSON 가공 (포맷팅/압축, 객체/배열 변환)
   - CSV/JSON 변환
   - 배열 처리

3. **데이터베이스**
   - SQL 쿼리 생성기 (INSERT, UPDATE, SELECT)
   - SQL 쿼리 분석기 (파싱, JSON 변환)
   - MyBatis Mapper 생성 (XML, 인터페이스)

4. **도구**
   - 타이머 JSON 생성
   - 색상 유틸리티
   - 타이머
   - QR코드 생성기

### 신규 제안 메뉴
5. **파일 가공**
   - **PDF 유틸리티**
     - PDF 텍스트 추출
     - PDF 이미지 추출
     - PDF 페이지 분할/병합
     - PDF 메타데이터 편집
     - PDF to 이미지 변환
     - 이미지 to PDF 변환
   
   - **이미지 가공**
     - 이미지 리사이징
     - 이미지 포맷 변환 (JPEG, PNG, WebP, SVG)
     - 이미지 압축 최적화
     - 이미지 회전/뒤집기
     - 이미지 필터 적용
     - 이미지 워터마크 추가
     - 이미지 메타데이터 편집 (EXIF)
     - 배치 이미지 처리
   
   - **문서 변환**
     - Excel to JSON/CSV 변환
     - Word 문서 텍스트 추출
     - Markdown to HTML 변환
     - HTML to Markdown 변환
     - 텍스트 파일 인코딩 변환
   
   - **아카이브 처리**
     - ZIP 파일 생성/압축 해제
     - 파일 압축률 비교
     - 여러 파일 ZIP 묶기

6. **네트워크/API**
   - **HTTP 유틸리티**
     - API 테스트 도구 (Postman 라이트)
     - HTTP 헤더 분석기
     - REST API 문서 생성
     - cURL 명령어 생성기
   
   - **URL/도메인**
     - URL 단축기
     - URL 분석기 (파라미터 추출)
     - 도메인 정보 조회
     - IP 주소 정보 조회

7. **개발자 도구**
   - **코드 변환**
     - JavaScript Minify/Beautify
     - CSS Minify/Beautify
     - HTML Minify/Beautify
     - 코드 하이라이터
     - 정규표현식 테스터
   
   - **해시/암호화**
     - MD5, SHA1, SHA256 해시 생성
     - Base64 고급 인코딩/디코딩
     - JWT 토큰 디코더
     - UUID 생성기
     - 패스워드 강도 체크
   
   - **개발 유틸리티**
     - Mock 데이터 생성기
     - Lorem Ipsum 생성기
     - 색상 팔레트 생성기
     - 아이콘 생성기
